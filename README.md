# 📦 Inventory Optimization Using Demand Forecasting

### End-to-End Supply Chain Analytics Project using Python, Statistical Forecasting & Power BI

---

## 📖 Project Overview

This project presents an end-to-end supply chain analytics solution to analyze inventory performance, optimize stock levels, forecast future product demand, and visualize business insights through interactive Power BI dashboards.

The project integrates Exploratory Data Analysis (EDA), Statistical Time Series Forecasting, and Business Intelligence to support data-driven inventory planning and supply chain decision-making.

---

## 🎯 Project Objectives

* Analyze inventory and sales performance.
* Identify demand patterns across products.
* Forecast future demand using statistical forecasting techniques.
* Evaluate inventory health and stock availability.
* Build an interactive Power BI dashboard for inventory monitoring and business insights.

## 📈 Statistical Forecasting

| Method                          | Purpose                            |
| ------------------------------- | ---------------------------------- |
| **Simple Moving Average (SMA)** | Forecast Future Product Demand     |
| **statsmodels**                 | Time Series Analysis & Forecasting |

### 📊 Forecasting Approach

The project applies statistical time-series forecasting using **Simple Moving Average (SMA)** with the **statsmodels** library to estimate future product demand based on historical sales data. The forecast supports proactive inventory planning and helps improve stock management decisions.
